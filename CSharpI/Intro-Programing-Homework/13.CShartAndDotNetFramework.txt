NET is an application development framework - it contains numerous libraries containing a range of functionality. The .NET libraries can run on the CLR and thus any language which can run on the CLR can also use 
the .NET libraries. In .NET you don't find only C#. You can find Visual Basic F#, Managed C++, IronRuby, and IronPython. You write code in the language of your choice to use the functionality in the .NET framework.
C# is a strong Object Oriented programming language that is mostly built on the .NET framework.
